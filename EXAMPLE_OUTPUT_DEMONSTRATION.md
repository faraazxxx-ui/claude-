**Claude API**
1. Sign up at https://console.anthropic.com
2. Generate API key
3. First API call:
```python
import anthropic

client = anthropic.Anthropic(api_key="YOUR_KEY")
message = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "Explain prompt engineering"}
    ]
)
print(message.content)
```

**OpenAI API**
1. Sign up at https://platform.openai.com
2. Generate API key
3. First API call:
```python
from openai import OpenAI

client = OpenAI(api_key="YOUR_KEY")
response = client.chat.completions.create(
    model="gpt-4",
    messages=[
        {"role": "user", "content": "Explain embeddings"}
    ]
)
print(response.choices[0].message.content)
```

**Perplexity API**
1. Sign up at https://docs.perplexity.ai
2. Generate API key
3. First API call:
```python
import requests

headers = {"Authorization": f"Bearer YOUR_KEY"}

    "model": "pplx-7b-online",
    "messages": [{"role": "user", "content": "Latest AI research"}]
}
response = requests.post(
    "https://api.perplexity.ai/chat/completions",
    headers=headers,
    json=data
)
print(response.json())
```

### Vertex AI RAG Setup

**Step 1: Enable Vertex AI**
```bash
gcloud services enable aiplatform.googleapis.com
```

**Step 2: Create RAG Corpus**
- Navigate to https://console.cloud.google.com/vertex-ai/rag/corpus
- Click "Create Corpus"
- Name: "research_documents"
- Description: "My research knowledge base"

**Step 3: Upload Documents**
- Prepare documents (PDF, TXT, HTML)
- Upload via console or API
- Wait for indexing (5-30 minutes)

**Step 4: Query with RAG**
```python
from google.cloud import aiplatform

aiplatform.init(project="YOUR_PROJECT")

# Query the RAG corpus
response = aiplatform.rag.retrieve_contexts(
    corpus="research_documents",
    query="What are the key findings in genomics research?",
    similarity_top_k=5
)

for context in response.contexts:
    print(f"Source: {context.source}")
    print(f"Content: {context.text}\n")
```

# 🆘 TROUBLESHOOTING GUIDE

## Common Issues & Solutions

### "Permission Denied" Errors
**Symptoms:** Cannot access GCP resources  
**Cause:** Insufficient IAM permissions  
**Solution:**
```bash
# Grant yourself necessary roles
gcloud projects add-iam-policy-binding PROJECT_ID \
  --member="user:YOUR_EMAIL" \
  --role="roles/owner"
```

### API Rate Limits
**Symptoms:** "429 Too Many Requests" errors  
**Cause:** Exceeding API rate limits  
**Solution:**
```python
import time

def api_call_with_retry(func, max_retries=3):
    for i in range(max_retries):
        try:
            return func()
        except RateLimitError:
            wait_time = 2 ** i  # Exponential backoff
            time.sleep(wait_time)
    raise Exception("Max retries exceeded")
```

### BigQuery Costs Too High
**Symptoms:** Unexpected charges  
**Cause:** Processing large datasets without limits  
**Solution:**
```sql
-- Always use LIMIT in development
SELECT * FROM dataset.table LIMIT 100

-- Use partitioning for large tables
SELECT * 
FROM dataset.table
WHERE DATE(timestamp) = CURRENT_DATE()
```
