   ```
   my-learning-context.md, create a personalized learning plan for [specific goal].
   ```
4. **Update as you progress** - your needs and knowledge will evolve!

```markdown
## Your Profile

1. Process 100GB+ VCF files automatically

3. Collaborate with bioinformaticians on shared code

## Current Challenges

**Feel Stuck On:** Setting up cloud environment, understanding which GCP service to use
**Most Frustrating:** Examples use web apps but I need to analyze genomics files

**Current Workflow:** Download files from sequencer → manually filter in Excel → copy-paste into analysis tool → export results
**Pain Points:** Can only process one patient at a time, prone to copy-paste errors, can't reproduce

## Immediate Needs

- Sample VCF files (can share 1GB subset)
- Previous Gemini chats about variant calling
- Google Drive with analysis templates

**Preferred Output:** Step-by-step checklist with exact commands, plus explanation of what each step does and why

## Success Criteria
**Week 1:** Upload files to Google Cloud and run simple query
**Month 1:** Process full cohort (50 patients) automatically  
**Month 2:** Build reproducible pipeline that team can use
```
