# AI OS Routing Card (March 2026)

## Default Flow
Speak → Capture → Ground → Build → Archive

## Routing Table

| Stage | If the job is... | Use this tool | Why |
|-------|-------------------|---------------|-----|
| CAPTURE | Messy thought | Wispr Flow / Superwhisper | Voice-to-text anywhere on desktop |
| CAPTURE | Quick note | Raycast Notes / Drafts | Instant text capture |
| CAPTURE | Voice memo | AudioPen | Rambling → structured text |
| THINK | Sharp framing | Grok | Direct, sharp, low-friction |
| THINK | Talk it out | Gemini Live | Spoken ideation with branching |
| CHECK | Need sources | Perplexity | Quick reality check with citations |
| CHECK | Deep research | Gemini Deep Research | Multi-step synthesis |
| BUILD | Docs/slides/output | Genspark | All-in-one creation |
| BUILD | Autonomous work | Manus (My Computer) | Local CLI execution |
| BUILD | Code changes | Claude Code | Reasoning + coding |
| REMEMBER | Meetings | Granola | Searchable meeting memory |
| LAUNCH | Desktop hub | Raycast | Front door to everything |

## Rules
1. One tool per stage
2. Do not research when it is time to build
3. Do not use Grok as final truth
4. Final outputs must be saved
5. If stuck, speak the problem out loud
