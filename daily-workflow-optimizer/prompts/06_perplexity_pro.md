# Perplexity Pro Search

## Optimized Prompt

```text
What is the optimal Personal Knowledge Management (PKM) methodology for a visual-spatial learner who needs to integrate handwritten notes from a Windows Surface, automate AI chat logs into a structured format with YAML headers, and implement the Eisenhower Matrix for task prioritization?

Focus on: A comparative analysis of methodologies like Zettelkasten, PARA, and Interstitial Journaling, evaluating their effectiveness in addressing the specified needs. The evaluation should consider compatibility with tools known for strong visualization and integration capabilities, such as Obsidian, Roam Research, and Logseq.

Prefer sources from: Academic research on cognitive science in note-taking, in-depth articles from established PKM expert blogs, and official documentation for software with robust API and plugin ecosystems.

If reliable sources cannot be found for a specific part of the query, state that clearly rather than speculating.
```

## Why This Works

This prompt is optimized for Perplexity Pro Search because it asks a single, precise question loaded with expert keywords (PKM, Zettelkasten, YAML), which guides the search engine effectively. It specifies a clear focus and preferred source types (academic, expert blogs), enabling Pro Search to leverage its real-time web access for a high-quality, cited answer. The final instruction acts as an anti-hallucination guardrail, ensuring the response is grounded in verifiable information found by the search.
