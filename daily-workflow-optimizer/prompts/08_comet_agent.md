# Comet Agent

## Optimized Prompt

```text
I need you to help me design a new, streamlined workflow for my daily notes and AI projects. You'll act as my research assistant to figure out the best way to connect my favorite apps and automate my processes.

Steps:
1. First, please research the integration capabilities of the following applications by visiting their official websites: Obsidian, Bear, Evernote, and NotebookLM. I need to know if they have APIs, webhooks, or other methods to connect with automation services like Zapier or Make.com.
2. Next, based on your research, analyze my current pain points. I want to connect handwritten notes from my Windows Surface (in OneNote/Whiteboard) to a digital daily note, and automatically save my AI chat outputs (from Claude, Gemini, Grok) into a central database like Notion, complete with metadata.
3. Then, propose a new, integrated workflow. This should recommend a central 'hub' application and explain how the other tools will connect to it. The goal is to automate as much as possible, from converting my drawings to text to cataloging my AI conversations.
4. Finally, create a template for my 'Daily Note' in Markdown format. It should include sections for 'What went well yesterday,' 'What I can improve,' 'Today's Goals,' and 'Today's Events.'

When done: Please provide a summary of your findings that includes the proposed workflow and the Markdown template for the daily note.

If you hit a wall: and can't find API or integration information for a specific app, make a note of it and suggest the best possible alternative for that part of the workflow.
```

## Why This Works

This prompt works well for Comet Agent because it breaks down a complex research and design task into a clear, sequential set of instructions. It uses natural, conversational language to delegate the task and defines a specific deliverable ('a summary of your findings'), which aligns with the agent's strength in executing multi-step browser-based research and content transformation.
